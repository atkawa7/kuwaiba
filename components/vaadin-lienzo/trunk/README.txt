Vaadin Lienzo Addon

 Enable LienzoPanel in Vaadin 7 via Lienzo Structure Graphics GWT Toolkit

Getting started
 See the demo application included.

Workflow

 Add the dependency
  <dependency>
   <groupId>com.neotropic</groupId>
   <artifactId>vaadin-lienzo</artifactId>
   <version>0.0.0</version>
  </dependency>
 
 GWT inherits
  <inherits name="com.neotropic.vaadin.lienzo.WidgetSet" />

Supported Features

 Add Lienzo Nodes

Supported Events

 Lienzo Component Mouse Over
 Lienzo Node Click
 Lienzo Node Right Click
 Lienzo Node Double Click
